# Flight Data Analysis

## Overview
This project analyzes airline customer activity and loyalty data using Python, Excel, and Power BI to uncover trends, generate business insights, and build an interactive dashboard.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Microsoft Excel
- Power BI

## Project Workflow
- Data Cleaning and Preprocessing using Python
- Exploratory Data Analysis (EDA)
- Interactive Dashboard Development using Power BI

## Key Insights
- Cleaned and prepared airline customer datasets for analysis.
- Analyzed customer behavior and identified key business insights.
- Built an interactive Power BI dashboard for data visualization.

## Project Files
- `Flight_Data_Analysis.ipynb`
- `Flight_Data_Analysis_Dashboard.pbix`
- `Customer Flight Activity.csv`
- `Customer Loyalty History.csv`
